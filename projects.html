React Chat

A full-stack superchat app using React as front-end and Firebase as back-end.

It's a single group chat room that everybody can participate in. To enter the chat room a user must login with their Google account. Once logged in they can post messages. Each message is a document in the fire store database. I contains the text, user ID and timestamp for that message. A query to the database is made for the most recent messages. Whenever the data changes on the back-end, the react app will update with the most recent messages on the front-end.
